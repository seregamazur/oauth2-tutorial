### 🔧 Installation

1. Clone the oauth2-tutorial repository:
```sh
git clone https://github.com/seregamazur/oauth2-tutorial
```

2. Change to the project directory:
```sh
cd oauth2-tutorial
```

3. Install the dependencies:
```sh
mvn clean install
```
